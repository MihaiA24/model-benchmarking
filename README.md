# PoC — Evaluación de modelos de IA para desarrollo

This repository hosts the autonomous coding-agent harness benchmark (Functional V1)
comparing OMP, OpenCode, and Hermes on reproducible real-code scenarios.

## Where to look

- [`CONTEXT.md`](CONTEXT.md) — authoritative project context and vocabulary.
- [`blueprint/`](blueprint/) — accepted contracts for the benchmark.
- [`docs/functional-v1-runbook.md`](docs/functional-v1-runbook.md) — how to run the Functional V1 benchmark.
